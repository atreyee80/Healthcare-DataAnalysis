# 🏥 Healthcare Data Analysis Dashboard

## 📌 Project Overview
This project analyzes healthcare patient data to uncover insights related to disease trends, patient demographics, hospital performance, and billing patterns.  

An interactive dashboard was created using Tableau to visualize key metrics and support data-driven decision-making.

---

## 🎯 Objective
- Analyze patient data to identify trends in diseases and demographics  
- Evaluate hospital performance and billing patterns  
- Build an interactive dashboard for easy data exploration  

---

## 🛠️ Tools & Technologies
- Python (Pandas) – Data cleaning and preprocessing  
- Tableau – Data visualization and dashboard creation  

---

## 📂 Dataset Features
The dataset includes the following columns:
- Name  
- Age  
- Gender  
- Blood Type  
- Medical Condition  
- Date of Admission  
- Discharge Date  
- Hospital  
- Doctor  
- Insurance Provider  
- Billing Amount  
- Room Number  
- Admission Type  
- Medication  
- Test Results  

---

## 🧹 Data Cleaning & Preparation
- Handled missing values and duplicates  
- Converted date columns to datetime format  
- Created a new feature: **Stay_Days** (hospital stay duration)  
- Created **Age Groups** for better analysis  

---

## 📊 Key KPIs
- Total Patients  
- Average Age  
- Total Billing  
- Average Stay Duration  
- Top Disease  

---

## 📈 Dashboard Visualizations
- Disease Distribution (patients by medical condition)  
- Age Group Distribution  
- Billing by Hospital  

---

## 💡 Key Insights
- The most common disease is **__Arthritis__**  
- Majority of patients fall in the **50 age group**  
- **Johnson PLC hospital** has the highest billing  
- Average hospital stay is **15days**
